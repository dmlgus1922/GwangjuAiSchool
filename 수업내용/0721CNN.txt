CNN - 컨벌루션 뉴럴 네트워크: 피쳐를 잘 뽑게 만들어준다

영상 인식: 영상 안의 물체를 인식하거나 분류하는 것
    영상은 이미지가 모인 것. 영상 처리 또한 이미지 처리.
컨벌루션 신경망은 영상 인식에만 사용되는 것이 아니지만 생물체의 영상 처리 구조에서 힌트를 얻었다.

데이터 전처리 -> 특징 추출 -> 분류
이미지가 입력 -> 엣지 추출 -> 엣지 집합 -> 물체 분류

컨벌루션 층을 지나면 주요 피쳐가 남는다.
 컨벌루션 -> 서브샘플링 -> 컨벌루션 -> 서브샘플링 ... -> 완전연결 출력
 컨벌루션: 같은 이미지를 여러개로 늘림
 서브샘플링: 쓸데없는 정보를 날려 사이즈를 줄임
 
컨볼루션 신경망에서는 하위 레이어의 노드들과 상위 레이어의 노드가 모인다.

입력층
    입력층에서 컨벌루션 연산을 통하여 특징을 뽑아내는 특징맵이 존재
    풀링 연산을 적용한다
    풀링 연산은 서브샘플링이라고도 하며 입력의 차원을 줄이는 연산이다
    특징을 모아주므로 공간의 제약 또한 줄여준다.

    컨벌루션 레이어와 풀링 레이어는 여러번 되풀이된다
    신경망의 맨 끝에는 완전히 연결된 구조의 전통적인 분류 신경망이 있어서 추출된 특징을 바탕으로 물체를 인식한다. 벡터 데이터가 나옴
    간단히 데이터 전처리기처럼 생각하면 된다.
    

컨볼루션은 주변 화소값들에 가중치를 곱해서 더한 후에 이것을 새로운 화소값으로 하는 연산이다.

풀링에는 맥스풀링, 민풀링이 있다
    맥스 풀링은 일정 영역에서 데이터가 가진 값 중 가장 큰 것만 남김
        - 이전의 컨벌루션으로 이전 데이터 정보가 반영되어 있다. 그 중 가장 주요정보라고 볼 수 있는 값이 가장 큰 데이터만 남기는 것
    민mean 풀링은 일정 영역에서 데이터가 가진 값의 평균을 남긴다.

풀링의 장점
    레이어의 크기가 작아지므로 계산이 빨라짐
    레이어의 크기가 작아진다는 것은 신경망의 매개변수가 작아진다는 것을 의미. 따라서 과적합이 나올 가능성이 줄어듦.
    공간에서 물체의 이동이 있어도 결과는 변하지 않는다. 물체의 공간이동에 대해 둔감해지는 것.

필터의 값은 미리 정해진 것이 아니고 학습된다. 오차를 통해 가중치를 바꿔가며 학습.
